FORTRAN(2003)

I'm not particularily sure why I did this one, I guess I had to satisfy some peverse curiosity. Anyway, FORTRAN is old, ugly, and largely still the standard for supercomputer benchmarking. It's not the worst possible language ever, but if you have the choice anything is signifigantly better.

Let's check out a simple hello world:


program hello
    implicit none
    print*, "Hello, World"
end program hello
